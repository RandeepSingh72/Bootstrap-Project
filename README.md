
# Bootstrap Project

I made this simple landing page using only Bootstrap and some custom css. This project is demo of my work.


## Optimizations

Box shadow of hamburger icon is removed.


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
